단계 로드맵 — 네이버 미팅이 첫 마일스톤

P0지금 ~ 7월 중순

제안용 데모 + 앱 선등록

네이버 미팅(7월 중순) 제출물: 웹 프로토타입(랜딩·캐릭터 갤러리·투표 목업) + AI 보컬 분석 데모(M2 — LVNEXUS API 검증 병행). 동시에 앱 껍데기 스토어 심사 접수. 본 문서 + 기능 스펙으로 LVNEXUS 합의.

P1접수·예선

ACT 1 — 캐스팅 오디션

M1·M2·M3(열람)·M4(응원 투표) 가동. 참가 접수 + AI 분석 리포트 공개 + 지원자 응원 젬 투표(라이트 BM). 참가 접수 기간 자체가 바이럴 구간(7/1). 피날레는 24캐릭터의 각성(캐릭터 첫 가동 텐트폴 모먼트) → 매력 스탯 동시 개방. ⚠ 각성 모먼트의 회차 배치는 편성 미결 — 7/6 레귤러 12회 스케일과 정합 필요.

P2본선

ACT 2 — 서바이벌·방송 연동

M4·M5·M6·M7 가동. 회차 투표 + 카테고리 투표 + 재화 BM 본격화 + 치지직 생방 연동 + 챗봇·오픈챗. 라운드별 평가축 전환 룰(보컬→방송→투표) 적용.

P3데뷔 이후

지분 정산 — 시즌 2 준비

M8 NFT 발행·등급 혜택 지급. 데뷔조 활동 연계(음원 RS·굿즈·인스파이어 공연). 동일 인프라로 시즌 2(프로듀스형) 포맷 전환.

미결정·선행 확인 사항

① LVNEXUS AI API
P0의 크리티컬 패스. I'M 앱 실물 품질 + API 제공 형태(동기/비동기, 처리 시간, 요금) 기술 검증 — 데모 성패 직결.
② 챗봇 업체
사명·API 조건·페르소나 커스텀 범위·미성년 안전 필터 확인 (7/2 교정본에서도 사명 미확정).
③ 인원·라운드 구조
24→16→10→5(7/1) vs 24→12→5(7/2 브리핑) 불일치 — 투표 엔진·화면 설계 전 확정 필수.
④ 노래 제출 방식
웹 파일 업로드 기본 확정 제안 — 앱 내 녹음은 음질·부정 방지 검증 후 P2. 미션곡 10곡의 권리 관계도 LVNEXUS 확인.
⑤ NFT 규제
거래 가능 디지털 굿즈는 가상자산 규제·스토어 정책(구글/애플 NFT 가이드) 법무 검토 선행 — 그래서 P3로 격리.
⑥ 웹 결제 정책
앱 내에서 웹 결제로 유도하는 동선은 스토어 정책 저촉 가능 — 결제 진입점 설계 시 정책 리뷰 필요.
⑦ 치지직 연동 범위
딥링크 수준인지, 방송 중 투표 오버레이·계정 연동까지인지 — 네이버 미팅 의제로 상정.
⑧ GATE 선발 배합
7/1 원칙은 "R1은 AI 개입 100%" — 팬 응원 투표를 결합하면 공정성 프레임과 충돌 여지. "AI 점수 = 자격(합격선), 팬 투표 = 합격자 중 선택" 절충안(§03) 기준으로 LVNEXUS·작가진과 배합 협의 필요.
⑨ 문화 수용성·팬 합성 클립
코어 버튜버 팬덤의 초기 반발(키즈나 아이 프레임)은 상수 — 개막 전 설계 헌법 매니페스토 공표로 선제 대응. 팬메이드 "후보 음성 × 캐릭터 비주얼" 합성 클립은 2차창작 가이드 금지 명시 + 모니터링 체계 필요(§03 방송 문법).
⑩ 투표 무결성·조작 프레임
프로듀스 조작 사태 이후 오디션 투표는 상시 의심받는 영역 — 다계정·앱 복제 어뷰징은 실전 사례 존재(§01-B). 디바이스 무결성 검증 + 투표 전량 원장 공개(검증 가능성)를 개발 필수 스펙이자 대외 신뢰 메시지로 이중 활용.
⑪ 각성 모먼트 편성 배치
24캐릭터의 각성은 각각 텐트폴 모먼트이나, 방송 회차로서의 편성은 미결 — 7/6 스케일(파일럿 8~10회 / 레귤러 12회, 회당 2~3시간)과 정합 필요. 회당 N캐릭터 각성 세그먼트 배치 / 일부 앱·라이브 이벤트 병행 등 옵션 검토. 방송 포맷 기획(작가 협업 범위) + LVNEXUS·치지직 협의 의제. ※ 구 표기 "각성 방송 24회"는 v0.5 작성 중 도입된 미확정 아이디어로, 확정 편성 아님(2026.07.07 정정).
