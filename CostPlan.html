비용 산출서 — A/B 이원 모드

Project AWAKE(각성) · 치지직 청구 기준 + 3사 투입 총괄 · v0.2 · 2026.07.13
STRICTLY INTERNAL — 원가 구조 노출 금지. 본 문서는 7/6 사전미팅의 "기획안 + 정확한 금액" 요구에 대한 내부 산출 원본이다. §02~03의 금액은 치지직 청구(제작비) 기준이며, 슈퍼스토리·LVNEXUS 투입분은 §06 3사 투입 총괄에 별도 계상한다 — 각 사의 투자가 각 사의 소유·회수와 1:1로 맞는 구조(v0.2 재편). 외부 제출 시에는 본 문서에서 파생한 요약(대분류 합계 수준)만 노출하고, 단가·수량 분해는 협상 카드로 보유한다. 모든 단가는 시장 단가 기준 내부 추정치 — ⚠ 표기 항목은 견적 검증 필수.
01

산출 전제

① 네이버 스튜디오 사용 — 스튜디오 렌탈비 제외 순수 제작비 (7/6 확인 전제. 스튜디오 운영 인력의 분담 범위는 미확정 — §05). ② 시즌 1회 기준(P0 준비 ~ P3 데뷔 인계, 약 6~7개월). ③ 데뷔 5인만 3D 승격 + 바디 골격 공용화(7/2 결론). ④ 캐릭터 24체 원화는 레이어 분리 + 마지막 붓 옵션 파츠 포함 발주. ⑤ LVNEXUS AI 심사 API 정산 방식(라이선스 vs 건당) 미확정 — 연동·검증 비용만 계상, 사용료는 §05 변수. ⑥ 앱은 슈퍼스토리 소유·개발 — 본 산출의 기술비는 시즌 투입분이며 자산 투자분과 구분 계리. ⑦ 귀속 원칙(v0.2): 시즌이 끝나도 남는 IP 자산(24체 원화·리깅, 앱 코어, 웹툰 콘텐츠)은 소유자인 슈퍼스토리가 투자하고, 방송을 위해 존재하는 것(생방 제작, 3D 승격 무대, 시즌 기술 투입, 리스크 관리, 마케팅)만 치지직에 청구한다 — "우리 돈으로 만든 캐릭터"라는 IP 양도 논리를 원천 차단하는 재무 구조.

두 모드의 정의 — 금액이 아니라 구조가 다르다
A안 · 표준 (25~30억) — 레귤러 12회 편성, 각성 24모먼트 캐릭터별 커스텀 연출, 별도 데뷔 쇼케이스(3D 첫 공개), 자체 마케팅 캠페인 정상 집행.
B안 · 압축 (20~25억) — 본선 10회 압축(각성 모먼트 회당 밀도 상향), 각성 연출 템플릿화(포맷 공통 + 캐릭터 스킨), 쇼케이스를 파이널 2부로 통합, 마케팅은 치지직 지면·트래픽 현물 분담 전제로 축소. B안은 "덜 쓰는 안"이 아니라 "치지직이 더 분담하는 안" — 협의 프레임 주의.

구 프레임(30~35억) 대비: 단가 현실화 + IP 자산의 슈퍼스토리 투자 이관으로 치지직 청구가 사업부 목표선(25~30 / 20~25) 아래로 내려옴 — 판이 줄어든 게 아니라 분담 구조가 정교해진 것(총 사업 규모는 §06). 제안서·PPT 표기 갱신 필요(§05).

06

3사 투입 총괄 — 각 사의 투자가 각 사의 회수와 맞는 구조

치지직 청구액(§02)은 전체 판의 일부다. 총 사업 규모는 3사 합산으로 유지되며, 각 주체는 자신이 소유·회수하는 것에 투자한다 — 이 정렬이 IP 원칙("이전 불가")과 RS 요구("투자의 회수")의 재무적 근거가 된다.
